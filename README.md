
# 🌍 Tourism Agency Web Application

## 📌 Overview
A full-stack web application simulating a travel agency platform. The system allows users to browse, book, and manage trips, with support for different user roles.

## 🛠️ Tech Stack
- Backend: ASP.NET Core
- Frontend: React
- Database: SQL Server
- ORM: Entity Framework
- Authentication: JWT

## 📷 Notes
This project was independently developed on my own initiative as a learning project to deepen my knowledge of .NET and React.
